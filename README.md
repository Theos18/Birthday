# Birthday Surprise Website

A beautiful birthday countdown website designed with a pink blossom theme.

## Features
- Countdown timer to 12 AM IST.
- Responsive design.
- "Surprise Sweetheart" button linking to a surprise page.

## How to Run
1. Open `index.html` in your web browser.
2. Or host it on GitHub Pages for online access.

## Customization
- Replace the image URL in `index.html` with your own photo.
- Update the message in `surprise.html`.
- **Music**: Download the song "Vhalam Aavo Ne" (or any song you like), rename it to `song.mp3`, and place it in the same folder as `index.html`.
